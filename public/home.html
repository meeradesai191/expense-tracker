DevOps Automation — Live Demo

An Expense Tracker app built to demonstrate a complete DevOps automation pipeline: source control, continuous integration, containerization and continuous deployment — all working together end to end.

Open the Expense Tracker →

App features

Add, edit and delete expenses · search and filter by category · sort by amount or date · export all data as CSV · set a monthly budget with a live over-budget warning · category-wise spend chart.

What this demo proves

🧑‍💻
Code pushed to GitHub
✅
GitHub Actions runs lint + tests automatically
🐳
App is packaged as a Docker image
🚀
Image is deployed live with zero manual steps

Every time new code is pushed to the main branch, this exact pipeline runs automatically — see it live on the Dashboard page, which shows the current build's git commit and deploy time pulled straight from the running container.

Tech stack

Node.js · Express · Jest (testing) · ESLint (linting) · Docker · GitHub Actions · Render (hosting)

DevOps Pipeline: Git → GitHub Actions (CI) → Docker (Build) → Deploy
